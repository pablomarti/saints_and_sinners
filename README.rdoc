== Saints and Sinners

Rails App template for implementing user management, for both, users and admins.

Admins are managed by levels, and using Devise with CanCan. Users are managed
with Devise and with Omniauth-Facebook.

The template uses Twitter Bootstrap and SimpleForm.


== Getting Started

1. Clone the project:
       git clone git@github.com:pablomarti/saints_and_sinners.git

2. Go inside of the project and execute bundle:
        bundle install

3. Edit the config/database.yml

4. Build the database and run de migrations

5. Execute the server
